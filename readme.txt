== Description ==

Navegg's Plugin for WordPress automatically insert our tag on every page of your website - eliminating your configuration work. Sign up on our website to get your Navegg ID, so we can start targeting your visitors.

O Plugin Navegg para WordPress coloca, automaticamente, a tag da Navegg em todas as páginas do seu site - eliminando seu trabalho de configuração. Faça seu cadastro em nosso site para ter acesso ao seu ID Navegg e comece a segmentar a audiência do seu site.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter your Navegg ID on field "Your Navegg ID" on menu Navegg
